# Hotel-Booking-Management-System
This project is a Hotel Management System web application developed using ASP.NET Web Forms, C#, SQL Server, and Bootstrap. The main objective of this project is to manage hotel room bookings, user registrations, and admin operations in a simple and efficient way. The system provides separate login modules for Admin and Users.

📌 Project Overview
<ul>
Developed a Hotel Management System web application to manage hotel room bookings and user registrations.
The system supports role-based authentication (Admin and User).
Admin can manage rooms and bookings, while users can view available rooms and make reservations.
Built as a real-time full-stack project to understand how frontend, backend, and database work together.
Focused on implementing secure login, CRUD operations, and database connectivity using ASP.NET.
</ul>

🛠️ Technical Stack & Implementation
<ol>
Frontend: ASP.NET Web Forms + Bootstrap
Backend: C#
Database: SQL Server
Database Connectivity: ADO.NET
Authentication: Session-based login system
Database Design: Tables for Users, Rooms, and Bookings
Stored Procedures: Used for Login, Insert, Update, Delete, and View operations
Development Tool: Visual Studio
</ol>
